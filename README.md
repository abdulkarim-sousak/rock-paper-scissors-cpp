# Rock Paper Scissors (C++)

Console-based Rock Paper Scissors game.

## Features
- Play against computer
- User can select number of rounds
- Displays final score at the end
- Ask player if they want to play again after finishing a game

## How to Play
1. Run the program
2. Enter number of rounds
3. Enter your choice each round: Rock / Paper / Scissors
4. See computer choice and score
5. Final score displayed at the end
6. After the game ends, choose whether to play again or exit

## Language
- C++
